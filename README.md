# womenapp-36
A smart personal productivity and safety web app that combines task management, wellness tracking, and emergency SOS features. It includes tools like a Pomodoro timer, daily journal, mood and water tracking, and an advanced SOS system with live location sharing, SMS/WhatsApp alerts, fake call escape, and siren alert for user safety.
# Women Safety & Productivity Web App

A smart web application that combines productivity tools with user safety features.

##  Features
- Task management system
- Pomodoro focus timer
- Daily journal & notes
- Mood and water tracking
- Emergency SOS system
  - Live location sharing
  - SMS & WhatsApp alerts
  - Fake call escape
  - Siren alert

## Technologies Used
- HTML
- CSS
- JavaScript

##  Purpose
This project is designed to improve daily productivity while ensuring personal safety using smart emergency features.
